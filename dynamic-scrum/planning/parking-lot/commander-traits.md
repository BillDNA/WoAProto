**Status:** not started · captured 2026-07-15 · graduates to a spec when the run shape + card system are settled

> Commander abilities that **bend the rules** — the run-defining pick at the top of Bill's Q.1 loop ("select commander").

Already in the post-V1 vision (CLAUDE.md): *"Commander abilities that bend the rules (e.g. guaranteed Conscription in the opening hand),"* with **side asymmetry** (different decks per side) behind it. This note accumulates trait ideas.

## Open questions to noodle
- **What can a trait bend?** Opening hand (guaranteed card), deck composition, piece stocks, a rules exception, an economy tweak?
- **How many, how strong?** A single run-defining pick vs. a small loadout. Asymmetry widens balance swings — the metrics/rubrics tooling (trustworthy-loop + actionable-data flags) has to land first.
- **Interaction with the deck:** does a commander bring a starting deck / side, or modify a shared one?
- **Measurability:** each trait is a balance variable. How does the loop grade an asymmetric matchup? The AI eval already can't see reserve-hoarding — asymmetry compounds the "bent ruler" risk.

## When it graduates
When the run shape (Q.1) and the card/deck system are settled enough to attach traits to. See `brainstorming` when ready.
