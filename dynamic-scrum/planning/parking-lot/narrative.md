# Parking Lot — Game narrative

**Status:** not started · captured 2026-07-15 · graduates to a spec / Roadmap when the run shape is decided

> Collect the story/framing for the roguelite — what the player is *doing* and why it holds together.

The battle prototype has no narrative frame yet. The steam-roadmap draft already points at a strong seam: the **field-journal / general's-diary aesthetic** — *"battles as journal entries, runs as campaigns in a general's diary."* This note accumulates narrative ideas until the run design (Q.1) is settled enough to frame them.

## Open questions to noodle
- What is the fiction of a **campaign** (Bill's term for a series of operations)? A war? A single general's tour of duty?
- Does narrative carry *between* battles (persistent stakes, named enemies) or stay flavor-only?
- How much story does a roguelite deck-builder actually want — framing vs. an authored arc?
- Lean into the aesthetic: the field-journal look is the differentiator; narrative should ride it, not fight it.

## When it graduates
When the run shape (Q.1 / [[run-design]]) is decided — narrative frames the run, so it follows the run design. See `brainstorming` when ready.
